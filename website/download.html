Sonata uses subversion for its development version control. You can checkout the latest code with:
     svn co http://svn.berlios.de/svnroot/repos/sonata/trunk sonata
You can also browse the latest code with WebSVN

Requirements

PyGTK 2.6.0 or newer
GTK+ 2.6.0 or newer
MPD 0.12 or newer (optional)
gnome-python-extras or PyGTK 2.10.x for system tray support (optional)
taglib and tagpy for editing metadata (optional)
ZSI for lyrics support (optional)
Python 2.5 or python-elementtree for audioscrobbler support (optional)
GCC (building only)
python-dev (on some distros; building only)
